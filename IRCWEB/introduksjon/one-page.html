Building a One-Page Site

In this lesson, we’ll explore the structure of a single-page layout — perfect for portfolios, event pages, or simple landing sites.

Lesson Overview

Unlike multi-page websites, a one-page site uses smooth scrolling and anchored sections instead of separate HTML files.

Tips for One-Page Design

  • Use a sticky navigation bar with anchor links (like #about, #services).
  • Make each section visually distinct.
  • Keep your content concise.
  • Optimize scrolling performance.

Homework

Create a simple one-page layout with a navigation bar linking to “Home”, “About”, and “Contact” sections.
